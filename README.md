# 暑期日计划

初一升初二暑期任务清单、知识地图、每日记录与家庭娱乐监督工具。

- 学生入口使用学生 PIN，可开始、暂停、自行恢复和申请结束娱乐。
- 暂停与恢复采用即时界面响应，云端在后台确认并校准记录。
- 系统只记录超时、长暂停和频繁暂停等客观事实，违规由家长最终判定。
- 家长可以修改当天已有的守时或违规判定，修正会追加到不可删除流水。
- 娱乐分为游戏、电影、其他娱乐三类；家务奖励通过后立即进入兑换池。
- 每日预习支持跨学科多选知识点，知识地图安排会追加到当日清单。\n- 网页由 GitHub Pages 托管，家庭数据通过云端接口实时同步。
